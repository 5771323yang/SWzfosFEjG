# 项目简介 python530

本项目是一款基于 Python + Django 的购物商城系统，包含完整开发文档。

## 技术栈

- **后端**: Python 3, Django
- **前端**: Vue.js
- **数据库**: MySQL 8
- **开发工具**: IntelliJ IDEA, Visual Studio Code

## 系统架构

- **架构模式**: MVC（模型-视图-控制器）
- **前后端分离**: 前后端分离开发，便于维护和扩展

## 环境配置

- **数据库**: MySQL 8
- **包管理器**: npm

## 操作系统

- Windows 10
- macOS

## 数据库可视化工具

- Navicat

## 功能模块

本项目主要包含以下功能模块：

1. 用户管理
2. 商品展示
3. 购物车
4. 订单管理
5. 支付系统

## 系统角色

- 用户
- 管理员

## 目录结构

```
project/
│
├── backend/                # 后端代码目录
│   ├── apps/               # django应用目录
│   ├── static/             # 静态文件目录
│   ├── templates/          # 模板文件目录
│   ├── manage.py           # 管理脚本
│   └── ...
│
├── frontend/               # 前端代码目录
│   ├── src/                # Vue源码目录
│   ├── public/             # 公共文件目录
│   ├── package.json        # 项目依赖和配置文件
│   └── ...
│
└── docs/                   # 项目开发文档
```

## 核心亮点

1. 前后端分离，易于维护和扩展
2. 详细的开发文档，降低学习成本
3. 灵活的系统架构，支持定制化需求
